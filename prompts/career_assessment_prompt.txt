# 任务
请根据以下学生信息，进行职业测评分析，为该学生提供个性化的职业规划建议。

# 学生信息
{student_info}

# 分析标准
1. 职业性格分析：
   - 性格特点：描述学生的性格特质
   - 适合的工作环境：分析适合的工作氛围和团队类型
   - 沟通风格：描述学生的沟通方式和人际交往特点

2. 职业兴趣倾向：
   - 感兴趣的领域：基于学生描述分析兴趣方向
   - 适合的职业类型：推荐匹配的职业方向
   - 行业趋势：相关行业的发展前景分析

3. 能力优势与提升建议：
   - 现有优势：分析学生已具备的能力和特长
   - 待提升能力：指出需要加强的技能领域
   - 发展建议：提供具体的能力提升路径

4. 职业目标规划：
   - 短期目标（1-2年）：大学期间或刚毕业的目标
   - 中期目标（3-5年）：职业发展的中期规划
   - 长期目标（5-10年）：长期职业发展愿景

# 输出要求
1. 基于学生信息进行客观分析
2. 使用积极正面的语言，鼓励学生
3. 结构清晰，层次分明
4. 建议要具体、可操作
5. 输出格式如下：

- **行业趋势分析**：[相关行业的发展前景]
