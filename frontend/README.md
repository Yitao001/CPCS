## 🚀 快速开始

### 前置条件

确保 API 服务已经启动：
- API 地址：http://localhost:9000
- 文档地址：http://localhost:9000/docs

### 打开前端页面

**方式一：直接双击打开**
直接双击 `frontend/index.html` 在浏览器中打开

**方式二：使用本地服务器（推荐）**

如果使用 VS Code：
1. 安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

或者使用 Python 简单服务器：
```bash
cd frontend
python -m http.server 8080
```
然后访问 http://localhost:8080

## 🔧 常见问题

**Q: 提示"加载岗位列表失败"？**
```
A: 确保 API 服务已启动，并且运行在 http://localhost:9000
```

**Q: 跨域错误？**
```
A: API 已配置 CORS 支持，请确保使用正确的 API 地址
```

**Q: 如何修改 API 地址？**
```
A: 编辑 app.js 文件，修改 API_BASE 变量
```
