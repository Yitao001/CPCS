# 大学生职业规划智能体 - API 调用指南

## 第一步：确认服务信息
- **服务地址：** `http://你的服务器IP:9000`
- **API Key：** `CPCS_test_API_key`

## 第二步：所有请求必须加的请求头
```
X-API-Key: CPCS_test_API_key
```

## 第三步：接口列表及调用示例

### 1. 获取岗位列表
**接口：** `GET /jobs?keyword=Java`

**JavaScript 调用：**
```javascript
fetch('http://你的服务器IP:9000/jobs?keyword=Java', {
  method: 'GET',
  headers: { 'X-API-Key': 'CPCS_test_API_key' }
})
.then(r => r.json())
.then(data => console.log(data))
```

**Python 调用：**
```python
import requests

url = 'http://你的服务器IP:9000/jobs'
params = {'keyword': 'Java'}
headers = {'X-API-Key': 'CPCS_test_API_key'}

response = requests.get(url, params=params, headers=headers)
print(response.json())
```

### 2. 解析简历
**接口：** `POST /resume/parse`

**请求体：**
```json
{
  "resume_content": "张三，大三，计算机专业，熟练掌握Java",
  "student_id": "student_001"
}
```

**JavaScript 调用：**
```javascript
fetch('http://你的服务器IP:9000/resume/parse', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'CPCS_test_API_key'
  },
  body: JSON.stringify({
    resume_content: '张三，大三，计算机专业，熟练掌握Java',
    student_id: 'student_001'
  })
})
.then(r => r.json())
.then(data => console.log(data))
```

**Python 调用：**
```python
import requests

url = 'http://你的服务器IP:9000/resume/parse'
headers = {'X-API-Key': 'CPCS_test_API_key'}
data = {
  'resume_content': '张三，大三，计算机专业，熟练掌握Java',
  'student_id': 'student_001'
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```
